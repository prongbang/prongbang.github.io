### AI-Upscaler คืออะไร

AI-Upscaler เป็นเครื่องมือสำหรับ **เพิ่มความละเอียดรูปภาพและวิดีโอ** โดยใช้เทคโนโลยี Real-ESRGAN ช่วยให้ภาพที่มีขนาดเล็กหรือรายละเอียดไม่คม สามารถ upscale เป็น 2x, 3x หรือ 4x ได้ง่ายขึ้น

ตัวโปรแกรมเหมาะกับงานที่ต้องการปรับคุณภาพไฟล์ media โดยไม่อยากเขียน command เองทุกครั้ง เช่น upscale รูปภาพสำหรับเว็บ, ปรับวิดีโอเก่าให้คมขึ้น, ทดลองโมเดล Real-ESRGAN หรือทำ batch งานผ่านหน้าแอปที่มี task queue ให้ดู progress ได้

ดาวน์โหลดได้จากหน้า Releases

{% highlight shell %}
https://github.com/prongbang/ai-upscaler/releases
{% endhighlight %}

---

### แนวคิดหลัก

* รองรับการ upscale รูปภาพ `png`, `jpeg`, `jpg` และ `webp`
* รองรับการ upscale วิดีโอแบบ process ทีละ frame
* เลือก scale ได้ เช่น 2x, 3x หรือ 4x
* มี Download Model Manager สำหรับดูและดาวน์โหลด model ที่ยังขาด
* มี Task Queue สำหรับดูงานที่กำลังประมวลผล, progress, preview และ output folder
* มี History สำหรับกลับไปเปิดผลลัพธ์ที่เคยทำเสร็จแล้ว
* เลือก GPU ได้ทั้ง Auto หรือระบุ GPU ID สำหรับ Vulkan processing
* ตั้งค่า output format, pixel format และ compression ได้

### วิธีใช้งานพื้นฐาน

#### 1. ดาวน์โหลดโปรแกรม

เข้าไปที่หน้า Releases แล้วเลือกไฟล์ portable executable ให้ตรงกับระบบที่ใช้งาน

* Windows
* Linux
* macOS

จากนั้นดาวน์โหลดและเปิดโปรแกรมได้เลย

---

#### 2. เลือกรูปภาพหรือวิดีโอ

เมื่อเปิดโปรแกรมแล้ว ให้เลือกไฟล์ที่ต้องการ upscale ก่อน โดยรูปภาพจะเหมาะกับงานที่ต้องการผลลัพธ์เร็ว ส่วนวิดีโอจะใช้เวลามากกว่า เพราะต้อง process เป็น frame-by-frame

ไฟล์รูปภาพที่รองรับ เช่น

{% highlight text %}
png
jpeg
jpg
webp
{% endhighlight %}

---

#### 3. ดาวน์โหลด model ที่จำเป็น

ถ้า model ยังไม่พร้อมใช้งาน สามารถเข้าไปที่ Download Model Manager เพื่อดู model pack ที่ติดตั้งแล้ว หรือดาวน์โหลด model ที่ยังขาดจากภายในแอปได้

ส่วนนี้ช่วยลดปัญหาที่เจอบ่อยเวลาใช้งาน Real-ESRGAN คือเปิดโปรแกรมแล้วหา model ไม่เจอ หรือต้องไปจัดการไฟล์เองหลายขั้นตอน

---

#### 4. ตั้งค่า upscale

ก่อนเริ่ม process สามารถตั้งค่าหลัก ๆ ได้ เช่น

* เลือก model ที่ต้องการใช้
* เลือก scale เช่น 2x, 3x หรือ 4x
* เลือก GPU เป็น Auto หรือระบุ GPU ID
* เลือก output format
* เลือก pixel format
* ตั้งค่า compression ถ้าต้องการลดขนาดไฟล์ output

ถ้าใช้งานครั้งแรก แนะนำให้เริ่มจาก scale 2x ก่อน เพื่อดูคุณภาพและเวลาในการประมวลผล จากนั้นค่อยปรับเป็น 4x เมื่อมั่นใจว่า model และเครื่องที่ใช้งานรองรับได้ดี

---

#### 5. เริ่มประมวลผลและดู Task Queue

หลังจากกดเริ่มงานแล้ว สามารถดู progress ได้ที่ Task Queue ซึ่งจะแสดงงานที่กำลังทำอยู่ พร้อม action ที่จำเป็น เช่น preview ผลลัพธ์, เปิด output folder หรือ cleanup งานที่ไม่ต้องการแล้ว

สำหรับวิดีโอ Task Queue จะมีประโยชน์มาก เพราะงานมักใช้เวลานานกว่ารูปภาพ และต้องติดตามความคืบหน้าเป็นระยะ

---

#### 6. ดูผลลัพธ์ย้อนหลังจาก History

เมื่อ upscale เสร็จแล้ว สามารถกลับไปดูงานที่เคย render ได้จาก History เพื่อเปิด preview หรือเปิด output folder ได้อีกครั้ง โดยไม่ต้องจำ path เอง

จุดนี้เหมาะกับ workflow ที่ทำหลายไฟล์ต่อเนื่อง เช่น upscale รูปหลายชุด หรือทดลองหลาย model เพื่อเปรียบเทียบผลลัพธ์

---

### ตัวอย่าง Workflow

สำหรับการ upscale รูปภาพทั่วไป flow จะประมาณนี้

{% highlight text %}
Select image
Download missing model
Choose model and scale
Select GPU and output format
Start upscale
Preview result
Open output folder
{% endhighlight %}

สำหรับวิดีโอ flow จะคล้ายกัน แต่ควรเผื่อเวลา process มากขึ้น เพราะต้องแยก frame, upscale แต่ละ frame แล้วประกอบกลับเป็นวิดีโอ

---

### เหมาะกับงานแบบไหน

* เพิ่มความละเอียดรูปภาพสำหรับเว็บหรือบทความ
* ปรับภาพเก่าหรือภาพขนาดเล็กให้คมขึ้น
* upscale วิดีโอที่ความละเอียดต่ำ
* ทดลองโมเดล Real-ESRGAN ผ่าน UI
* ทำงานหลายไฟล์แล้วต้องการดู progress ผ่าน Task Queue
* เก็บประวัติผลลัพธ์ไว้เปิดดูภายหลัง

---

### ข้อควรรู้

* การ upscale ไม่ได้ทำให้รายละเอียดที่ไม่มีอยู่จริงกลับมาสมบูรณ์เสมอไป คุณภาพผลลัพธ์ขึ้นอยู่กับไฟล์ต้นฉบับและ model ที่เลือก
* scale ยิ่งสูง ยิ่งใช้เวลาและทรัพยากรมากขึ้น
* วิดีโอใช้เวลามากกว่ารูปภาพ เพราะต้องประมวลผลหลาย frame
* ถ้าใช้งาน GPU ควรเลือก Auto ก่อน แล้วค่อยระบุ GPU ID เมื่อจำเป็น
* ควรทดลอง output format และ compression กับไฟล์ตัวอย่างก่อนใช้กับงานจริงจำนวนมาก

### สรุป

AI-Upscaler เป็นเครื่องมือที่ช่วยให้การใช้งาน Real-ESRGAN ง่ายขึ้นมาก เพราะรวมขั้นตอนสำคัญไว้ในแอปเดียว ตั้งแต่เลือกไฟล์, ดาวน์โหลด model, ตั้งค่า scale, เลือก GPU, ดู progress ใน Task Queue ไปจนถึงเปิดผลลัพธ์ย้อนหลังจาก History

ถ้าต้องการเพิ่มความละเอียดรูปภาพหรือวิดีโอโดยไม่อยากจัดการ command และไฟล์ model เองทุกครั้ง ตัวนี้เป็นอีกเครื่องมือที่ใช้งานได้สะดวกและเหมาะกับงานทดลองหรือใช้งานจริงได้ทันที
